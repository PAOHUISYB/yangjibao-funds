# 基金 ID 映射表示例

> 以下为示例数据，请替换为你自己的基金持仓 ID。
> 可通过 `GET /search_fund?keyword=基金名称` 接口查询获取 fund_id。

## API 请求体示例（公网行情批量查询）

```json
{
  "funds": [
    {"fund_id": 1359, "data_source": "1"},
    {"fund_id": 20864, "data_source": "1"}
  ]
}
```

## 字段说明

| 字段 | 说明 |
|------|------|
| `fund_id` | 养基宝内部基金 ID（用于公网行情查询） |
| `data_source` | 固定值 `"1"` 表示使用主数据源 |
| `code` | 基金交易代码（6位，支付宝/天天基金同款） |

## 常见基金类型说明

| 类型 | 特点 | 净值更新 |
|------|------|---------|
| A股 ETF 联接 | 跟踪国内指数（沪深300、中证500、AI等） | T日当晚 19:00~22:00 |
| QDII 基金 | 投资海外（纳指、标普、港股等） | 滞后 2~3 个交易日 |
| 混合型 | 主动管理，股债灵活配置 | 同 A股 ETF 联接 |
| 黄金ETF联接 | 跟踪黄金现货价格 | T日当晚 |

## QDII 净值滞后说明

QDII 基金由于涉及跨境换汇，净值公布比A股基金晚 2~3 个交易日。

例如：
- 周一买入纳指基金 → 实际按周一收盘净值成交
- 查询时会显示 `true_valuation_date`，以该日期为准计算涨跌
- 盘中显示的 `vgszzl` 为估值涨幅，非真实净值涨跌
